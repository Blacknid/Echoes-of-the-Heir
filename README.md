# Michi-s-adventure

#Michi's adventure este un joc RPG programat in limbaj nativ java

## System Requirements

### Minimum (play)
- OS: Windows 10 (64-bit) or newer
- CPU: Dual-core processor (Intel i3 4th gen / AMD equivalent)
- RAM: 4 GB
- GPU: Integrated graphics capable of Java2D/Swing rendering
- Storage: 300 MB free space

### Recommended (smooth 60 FPS)
- OS: Windows 10/11 (64-bit)
- CPU: Quad-core processor
- RAM: 8 GB
- GPU: Any modern integrated or dedicated GPU
- Storage: SSD with 300 MB+ free space

## Java Requirements

### For players (packaged .exe)
- No separate Java install is typically required when using the packaged installer/runtime.

### For developers (run/build from source)
- JDK 17 or newer
- `jpackage` available (for generating Windows `.exe` builds)
